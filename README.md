# Слайдер изображений «Горизонты»

Учебный проект: адаптивный зацикленный слайдер на чистых HTML, CSS и JavaScript.

## Возможности

- переключение изображений кнопками «вперёд» и «назад»;
- зацикливание в обоих направлениях;
- отображение текущего номера изображения;
- управление стрелками клавиатуры;
- адаптивный интерфейс и поддержка `prefers-reduced-motion`;
- локальные изображения — проект работает без подключения к интернету.

## Запуск

Откройте файл `index.html` в браузере. Дополнительная установка не требуется.
